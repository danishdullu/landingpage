How does a Protected Portfolio work?

A Protected Portfolio is an FDIC insured bank deposit combined with market performance. The bank deposit is a CD provided by your bank while the market performance is driven by an advanced portfolio of stocks, bonds, and other assets.

Earn 5.25 times average CD Return

Protected Portfolio earns an average annual return of 4.2%. Meanwhile the national average 1YR CD return is 0.80%. Given that the Protected Portfolio return is linked to the markets, scenarios exist where the return can be less than the average 1YR CD return. Likewise, scenarios exist where the Protected Portfolio can outperform traditional CDs far more than 5.25 times

How is a Protected Portfolio 100% principal protected?

The invested amount purchases an FDIC insured deposit. Only the interest is invested in the market linked portfolio.

How to earn an average return of 15% or more?

Protected Portfolio includes an innovative referral program that allows clients to increase their average return without sacrificing the safety of principal protection nor the FDIC insurance.
